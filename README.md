# Weather Bot

Weather Bot — это консольное приложение на C++, которое анализирует погодные данные через API и даёт рекомендации, что надеть в зависимости от температуры, осадков и других погодных условий.

## Возможности

- Получение текущей погоды через API.
- Анализ погодных данных.
- Советы по выбору одежды (легкая, теплая, дождевик и т.д.).
- Простая консольная реализация.
